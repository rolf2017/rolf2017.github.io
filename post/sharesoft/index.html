一些不错的软件

2018-01-15
Soft

Contents

    • Windows
    • Mac
    • Opensource

   从ghost到纯净版装机，从XP到Mac，分享一些目前正在使用的软件。

Windows

   从XP到Windows10，折腾的过的软件太多了。Photoshop， Office就不上链接了，win7使用office2016，win10就用office2019吧。

  • 杀毒软件： 火绒 #难得的国产良心之作，感觉是目前最好的杀毒软件了，免费
  • 系统维护： Glary Utilities #有很多功能，可修复注册表，各种清理，整理，备份等等，可自行尝试激活Pro版本
  • 浏览器： 火狐 #以前使用的是谷歌，现在感觉这个更好一点，有实力可以避免.cn版本
  • 输入法： QQ输入法 #腾讯除了QQ和微信，输入法也可以使用，无广告弹窗
  • 科学计算： Spyder #好用的Python发行版本，也可直接安装anaconda
  • 音乐: 网易云音乐 #不知道是听的多还是什么原因，感觉更喜欢网易云推荐的歌单，虾米也在使用
  • 视频解码： K-lite Code Pack #视频解码选择这款，使用Windows media player播放
  • 文本编辑： UltraEdit #强大的功能，可查看编辑各种格式，需要实力激活
  • 百度云下载： PanDownload #百度云下载利器，有实力还是要买百度云超级会员
  • 刻盘工具： PowerISO #以前用的另外一款，一次制作Linux启动盘时只有这款成功

Mac

   Mac入手，折腾继续。

  • Shell： Oh-my-zsh #bash看久了就想换个，然后就停不下来了
  • 软件管理： Homebrew #之前使用Macport，已经换成了Homebrew
  • 解压缩： The Unarchier #比较好用的解压缩软件
  • PDF： Skim #免费
  • 绘图： GMT #免费开源绘图软件
  • 代码编辑器： Sublime Text #可无限期试用，也可用Xcode，看个人喜好
  • 云笔记： Simplenote #简单轻便好用的云笔记，好记性不如烂笔头，真心好用
  • 英语词典： 欧陆词典 #Mac下非常好用的词典，使用Pro版本添加牛津和Collins词库非常非常好用
  • 安全防护： OverSight #保护隐私的意识还是要有的，实时监控摄像头和麦克风，会进行提示通知

Opensource

   上面有一部分的软件已经属于自由软件，下面再单列一些自由软件，都非常棒。

  • 录屏软件： OBS #很好用的录屏软件；据说很多人用来直播，没有体验过
  • 图像处理： GIMP #可替代Photoshop的软件
  • 解压缩： 7Z #比较好用的解压缩软件
  • 下载： Aria2 #可以代替迅雷的存在
